3.2 DATA MODELS (Minimal)

class MarketNoteLine:
    timestamp: str      # "9:49"
    raw_text: str       # Full line preserved
    tickers: List[str]  # Extracted: ["NKE"]
    is_trade: bool      # Pattern match: "long/short TICKER @ PRICE"
    trade: TradeInfo?   # If is_trade: direction, ticker, price, size, reason

class DailyReport:
    date: date
    market_notes: List[MarketNoteLine]  # Organised, timestamped
    trades: List[TradeInfo]              # Extracted from market notes
    highlights: List[str]                # User-marked or LLM-flagged for carry-forward

class PreMarketNotes:
    date: date
    carry_forward: List[str]      # From prior day highlights
    economic_events: List[Event]  # FOMC, CPI, earnings
    user_rules: List[str]         # Active approved rules

class WeeklyReview:
    week: str
    daily_reports: List[DailyReport]
    pattern_suggestions: List[PatternSuggestion]
    metric_tracking: Dict[str, MetricSeries]  # User-approved metrics

class PatternSuggestion:
    pattern: str                    # "Tall bounces on shorts = avoid"
    supporting_days: List[date]
    evidence_quotes: List[str]      # Your exact words from Market Notes
    user_confirmed: bool = False    # You confirm/dismiss

class MonthlyTrajectory:
    month: str
    weekly_summaries: List[WeeklyReview]
    high_level_notes: str           # Broad goals, direction

5. LLM PROMPTS (Core Intelligence)
================================================================================

5.1 DAILY_REPORT PROMPT
-----------------------
INPUT: Raw Market Notes text (timestamped lines)

TASK: Parse into structured JSON. DO NOT summarise, judge, or infer.
Keep EVERY line. Only structure it.

OUTPUT:
{
  "market_notes": [
    {"timestamp": "9:45", "raw_text": "watching NKE short - D1 breakdown", "tickers": ["NKE"], "is_trade": false},
    {"timestamp": "9:49", "raw_text": "FOMO entry NKE short - didn't wait for SPY confirmation", "tickers": ["NKE"], "is_trade": false},
    {"timestamp": "9:49", "raw_text": "short NKE @ 55.10, 4 shares - D1 breakdown", "tickers": ["NKE"], "is_trade": true, "trade": {"direction": "short", "ticker": "NKE", "price": 55.10, "size": 4, "reason": "D1 breakdown"}},
    {"timestamp": "10:16", "raw_text": "exit NKE @ 54.97 - SPY closed above VWAP+", "tickers": ["NKE"], "is_trade": true, "trade": {"direction": "exit", "ticker": "NKE", "price": 54.97, "size": 4, "reason": "SPY closed above VWAP+"}}
  ],
  "highlights": [
    "Watch NKE for follow-through short if SPY confirms lower",
    "Rule: NO short entries on gap days without M5 pullback confirmation",
    "Bias: LPTE environment, favour shorts but need confirmation"
  ]
}

RULES:
- Trade pattern: "(long|short) TICKER @ PRICE, SIZE shares? - REASON"
- Exit pattern: "exit TICKER @ PRICE - REASON"
- Extract ALL tickers mentioned per line
- Highlights = actionable carry-forward items (watchlist, rules, bias)
- NO analysis, NO "you should have" - pure extraction

5.2 PRE_MARKET PROMPT
---------------------
INPUT: Today's DailyReport + configured economic calendar + active user rules

TASK: Generate brief pre-market notes for tomorrow

OUTPUT:
{
  "carry_forward": [
    "NKE: Watch for follow-through short if SPY confirms lower",
    "Bias: LPTE - favour shorts with confirmation",
    "Rule: NO short entries on gap days without M5 pullback"
  ],
  "economic_events": [
    {"time": "10:00", "event": "CPI YoY", "impact": "HIGH"},
    {"time": "14:00", "event": "FOMC Minutes", "impact": "HIGH"}
  ],
  "user_rules": [
    "NO FOMO entries - wait for M5 pullback close",
    "Don't exit winner on single M5 candle"
  ]
}

RULES:
- Max 5 carry-forward items
- Only major economic events (FOMC, CPI, NFP, GDP, major earnings)
- User rules = approved rules from weekly review
- Brief. One screen-readable in 60 seconds.

5.3 WEEKLY_PATTERNS PROMPT
--------------------------
INPUT: 5 DailyReports (market_notes + highlights)

TASK: Find recurring patterns in YOUR words. Suggest - don't conclude.

OUTPUT:
{
  "pattern_suggestions": [
    {
      "pattern": "Tall bounces on shorts = avoid shorts",
      "supporting_days": ["2026-03-24", "2026-03-31"],
      "evidence_quotes": [
        "not many leakers today - lots of tall bounces on shorts",
        "today was just not a good day for shorts - many tall bounces"
      ]
    },
    {
      "pattern": "FOMO entries on gap days without confirmation",
      "supporting_days": ["2026-03-12", "2026-03-13", "2026-03-31"],
      "evidence_quotes": [
        "was a FOMO entry on NKE. Didnt wait for market confirmation",
        "FOMO and panic out, Dave gets beautiful short",
        "chased IREN flush at 10:50 without confirmation"
      ]
    }
  ]
}

RULES:
- Pattern = YOUR phrasing where possible
- Minimum 2 supporting days
- Evidence = exact quotes from your Market Notes
- NO implied rules - just pattern observation
- You decide what becomes a rule in weekly review
