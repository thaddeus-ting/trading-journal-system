CATEGORIZATION RULES:
- "observation": Market watching, analysis, no action taken
- "decision": Deliberate choice to wait, skip, hold
- "entry": Opening a position (long/short)
- "exit": Closing a position
- "emotion": FOMO, fear, frustration, greed explicitly mentioned
- "skip": Consciously decided not to trade a setup

KEY LEVELS TO EXTRACT:
- VWAP, VWAP+, VWAP-
- D1 SMAs: 8, 15, 50, 100, 200 EMAs
- M5/M15 levels: 8/15 EMA, VWAP
- PDH, PDL, PWClose, Friday HOD/LOD
- ATRH, ATRL
- Prior day LOD/HOD

RULES:
1. Preserve EXACT language from notes - do not paraphrase
2. If a line has no timestamp, estimate from context or use "?:??"
3. Match entries to exits by ticker and sequence
4. Highlights = actionable items for tomorrow (watchlist, rules, bias)
5. Market bias = your stated or implied view from the notes
6. If Market Notes empty, return empty arrays

HIGHLIGHTS GENERATION (highlights_for_carry_forward):
- ONLY include points from notes that have tags (#flag, #gold, #sector, #watch, #review, #todo, #ptrade, #mistake, #good, #bad, #emotion, #setup)
- For EACH tagged note, create ONE brief sentence (max 1 sentence) capturing the key point
- Prioritize #flag, #gold, #sector tags - they MUST be included
- Keep highlights concise and action-oriented
- Format: "TICKER/TOPIC: 1-sentence summary of the tagged point"
- Order highlights by TIME (earliest timestamp first) as a bullet list
